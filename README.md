# NestJS - Prisma Blog API
NestJSとPrismaで作成したブログ用のAPIです。  
APIの勉強のために作成しました。  
デプロイにはrenderを使用しているので読み込みに30秒ほど時間がかかります。  
また画面はなくAPIのみなのでSwaggerUIのみになります。  

https://nest-prisma-blog-api.onrender.com/api
